# Skills

Use this file to track Cursor skills you want the agent to use proactively.

## Preferred skills

Add one per line in the format:

- owner/repo — when to use it

Examples:

- vercel-labs/agent-browser — UI checks and screenshots
- vercel-labs/agent-skills (vercel-react-best-practices) — React best practices
- vercel-labs/skills (find-skills) — discover skills to add
